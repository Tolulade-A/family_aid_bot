# Family_Aid_Bot
A conversational bot built using DialogFlow

## **Project Overview**

The goal of this project was to design and build a chatbot that could provide information on family law topics such as divorce, separations, and child support for cases in Country X. The chatbot was designed using DialogFlow and was intended to be a resource for individuals seeking information on these topics.

### Scope of the Project (v1):

- Research and gather information on family law topics such as divorce, separations, and child support in Country X.
- Design and build a chatbot using DialogFlow that can provide information on these topics to users.
- Test and refine the chatbot to ensure it is able to accurately understand and respond to user questions.
